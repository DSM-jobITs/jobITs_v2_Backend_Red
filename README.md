## 기능
- 사용자 인증
  - 대덕소프트웨어마이스터고등학교에 재학 중인 학생인가?

  - 등록된 채용 의뢰에 대한 정보 조회
- 관리자
  - 새로 접수된 채용 의뢰 등록
  - 기존에 등록된 채용 의뢰의 수정/삭제
